# sakura renderer web

官网的源代码，可以在本地部署查看文档和渲染

## 文章源代码

官网所有的文档的源代码位于src/Document/documents路径下

文件命名方式与官网地址路径的最后一个单词相同，如：官网根地址/docu/a下显示的文档，在源代码中保存在a.js下


## 官网链接

## 渲染器源代码链接
See [Sakura Renderer](https://github.com/SakuraLong/sakuraRenderer).
